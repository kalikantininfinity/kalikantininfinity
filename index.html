Flagship Project

AMDVS — Autonomous Multi-Domain Vehicle System
● Active Build

An autonomous ground rover built from a bare board — no lab, no grant, no team. Phase 1 achieves real-time object detection on-device. Phase 2 adds a cooperating drone that scouts ahead and docks back on the rover: the Perseverance + Ingenuity idea, built in a bedroom. The goal isn't a demo that looks smart — it's a system that stays smart when a sensor lies, the sun blinds the camera, or the path disappears.

How I Build

Builder before anything else. I learn by making the thing, breaking the thing, and rebuilding it better. Every frame of 30 FPS was earned by something breaking first.

Safety is not deferred. The unglamorous layer — safety reflexes, sensor fusion, the code that keeps an autonomous system alive when reality stops cooperating — that's the interesting work. A hardware e-stop under 50ms is a from-the-start requirement, not a future feature.

Each project is a brick. Reusable, documented, compounding into a stack I own end to end. No black boxes I can't explain or rebuild.

Self-funded means every decision counts. No lab, no grant, no team. That constraint is a feature — it forces prioritization, documentation, and real engineering over demos.
